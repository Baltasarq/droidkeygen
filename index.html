DroidKeyGen

Esta aplicación permite generar, de manera muy sencilla, claves auto firmadas para firmar aplicaciones Android (entre otras cosas). Permite generar una clave con los datos personales, así como comprobar una clave ya creada.

This application simply generates self-signed keys for, among others, Android apps. It is able to create a key store with your personal data, as well as checking an already existing key.
